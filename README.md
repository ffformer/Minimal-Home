# Minimal Home — 商品展示页面

一个简洁现代的家具电商商品展示页面，使用原生 HTML / CSS / JavaScript 构建。

## 快速启动

直接在浏览器中打开 `index.html` 即可运行，无需构建工具或依赖安装。

## 项目结构

```
商品展示/
├── index.html          # 主页面（语义化 HTML5）
├── styles.css          # 全部样式（设计令牌 + 组件 + 响应式）
├── script.js           # 交互逻辑（渲染 / 筛选 / 购物车 / 菜单）
├── Product data.json   # 商品数据源（参考用）
├── gen-img-*.png       # Logo 图片
└── README.md
```

## 页面组成

| 区域 | 内容 |
|------|------|
| **Header** | Logo + 导航菜单（首页 / 商品 / 关于 / 联系）+ 购物车入口 + 移动端汉堡按钮 |
| **Banner** | 全宽 Hero 区域，背景图 + 半透明遮罩 + 标题 / 副标题 / CTA 按钮 |
| **商品列表** | 标签筛选栏 + CSS Grid 卡片网格，JS 从数据动态渲染 |
| **Footer** | 品牌信息 + 链接导航（购物指南 / 政策） |

## 商品卡片

每张卡片包含：商品图片、标题、价格、标签（New / Hot / Sale 颜色区分）、加入购物车按钮。全部由 `<template>` 克隆 + JS 数据驱动渲染，**不手写任何重复 HTML**。

## 页面交互

实现了 **6 项交互**（要求至少 2 项）：

| # | 交互 | 实现方式 |
|---|------|---------|
| 1 | 卡片 hover 动效 | CSS transform + box-shadow，`@media (hover: hover)` 触摸设备跳过 |
| 2 | 加入购物车 + Toast 反馈 | 事件委托 + 动态创建通知组件，2.8s 自动消失 |
| 3 | 移动端菜单展开 / 收起 | 汉堡图标 → X 动画，遮罩点击 / 链接点击 / Esc 键多路径关闭 |
| 4 | 商品标签筛选 | 数据过滤 + DOM 重新渲染 + 按钮状态同步，空结果友好提示 |
| 5 | 购物车系统 | 全屏界面，支持多数量、勾选、全选、数量 ±、删除、结算 |
| 6 | 一键回到顶部 | 超过 400px 滚动后浮现，平滑滚动 |

## 响应式设计

Mobile-first 策略，4 个断点：

| 断点 | 列数 | 导航 | 备注 |
|------|------|------|------|
| < 640px（手机） | 2 列紧凑 | 汉堡菜单 | 卡片小字号、紧凑间距 |
| 640–899px（平板竖屏） | 2 列舒适 | 汉堡菜单 | 卡片逐步放大 |
| 900–1023px（平板横屏） | 3 列 | 汉堡菜单 | 标题左对齐 |
| ≥ 1024px（桌面） | 3–4 列 | 完整横向导航 | 购物车右面板化 |

## 设计决策

### 配色
暖色中性调，强调家庭与家具的温暖感：
- 页面底色 `#faf9f7`（暖白）
- 文字 `#2b2722`（暖黑）
- 强调色 `#b8753e`（铜色），仅用于 CTA、激活态筛选、链接
- 标签：New 绿、Hot 橙、Sale 红

### 排版
系统字体栈 `system-ui` + 中文字体回退，零外部依赖。层级通过字号和粗细对比区分，Banner 标题使用 `clamp()` 流体缩放。

### CSS 架构
- 所有视觉参数集中为 `:root` 内 CSS 自定义属性（设计令牌）
- BEM 命名规范，无样式冲突
- `auto-fit + minmax` 实现断点无关的网格自适应
- `@media (hover: hover)` 区分触摸与鼠标交互

### JavaScript 架构
三层分离：**Data → Render → Interactions**

```
PRODUCTS[] → renderProductGrid(filter) → createProductCard(product)
                                              ↓
state { cart, currentFilter, menuOpen } ← 交互事件
```

- 事件委托用于商品列表和购物车列表（容器级监听，避免重新绑定）
- `<template>` 克隆渲染卡片，性能优于字符串拼接
- 购物车状态集中管理，渲染函数幂等
